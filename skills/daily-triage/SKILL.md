# Skill — Daily Triage

**Trigger:** George asks to plan the day or process his inbox.

## Instructions
1. Call `list_items` (status=draft) and `get_inbox_summary` to see what's pending.
2. Rank by priority (urgent → high → medium) and by deadline/age.
3. Surface the **ONE** thing that would make today a win (per `identity/core.md` — reduce overwhelm).
4. Propose a short, ordered plan (3–5 items max). Don't dump the whole list.
5. For anything George approves, use `update_item` to set status/priority; capture new ideas with `create_item`.
6. If patterns emerge (e.g. repeated stalls), log via `log_intelligence`.

## Output style
Lead with the ONE focus, then a tight ordered list. Warm, direct, no overwhelm.
