# Skill — AGI Build Sprint

**Trigger:** A focused build/work session on the AGI Dashboard (or any project build block).

## Instructions
1. Read `identity/core.md` and the target `projects/[project]/context.md`. Skip unrelated projects.
2. State the ONE objective for this block in a single sentence. Confirm with George.
3. Pull current state from `context.md` + the live DB (inbox items, recent intelligence).
4. Work in the smallest shippable increments. After each: verify it actually works (build/test/preview), don't claim done on faith.
5. If George drifts to a new idea, capture it to the inbox and re-anchor to the objective.
6. Protect momentum: finish the 80%-done thing before starting new.

## On close (always)
- Write a session file to `projects/[project]/sessions/YYYY-MM-DD-[block].md` (use `sessions/_TEMPLATE.md`).
- Append any durable insight to `intelligence/breakthroughs.md` or `patterns.md`.
- Mirror the record into the DB (`intelligence_logs` / `memory_entries`) so the live agent recalls it.
- Update `projects/[project]/context.md` "Current state" + "Next checkpoints".

## Output style
Concise, direct. Lead with what shipped and the next single action.
