# Skill — Capture Voice Note

**Trigger:** A new voice note arrives, or George asks to process / pull from his voice notes.

## Instructions
1. Use `search_voicenotes` (live) for the relevant query/tags, or `sync_voicenotes` to bulk-pull recent notes.
2. For each note: extract the core idea, decisions, and any concrete actions.
3. Classify (per `routing/mind-vs-money.md` and `routing/personal-vs-work.md`) and route to the right project.
4. Save durable thinking with `create_memory` (type=insight/note); turn concrete actions into `create_item` (inbox) — but confirm before creating many.
5. Don't dump raw transcripts — summarize. Strip filler.

## Output style
Per note: 1-line summary + extracted actions. Ask before bulk-creating inbox items.
